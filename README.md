# カメラ撮影統計アナライザー

パソコン内の画像フォルダを読み込み、写真のメタデータ(EXIF)から撮影傾向をグラフ化するツールです。
次に買うべきレンズの検討や、よく使う焦点距離・F値・ISO感度の把握に役立ちます。

## 主な機能

- 焦点距離、F値、シャッタースピード、ISO感度、撮影時刻などのグラフ表示
- 指定したカメラ本体やレンズによるデータの絞り込み
- JPEGファイルのほか、主要なRAWデータの読み込み
- 絞り込んだ分析データのCSV出力

## 使い方

1. Releasesページから最新の CameraStatsAnalyzer.exe をダウンロードします。
2. ダウンロードしたファイルを実行してアプリを起動します。
3. 画面の「追加」ボタンから画像が入っているフォルダを選択します。
4. 「解析開始」をクリックするとデータがグラフ化されます。
